Installation

Just git clone https://github.com/mikelsv/mglcore.git
Create a folder in the root and create your first project with mglcore!

index.html

Create index.html with code: <!DOCTYPE html> <html> <head> <title>My Game</title> </head> <body> </body> </html> <script src="../mglcore/mgl.package.js"></script> <script>mglPackageInit("main.js");</script> The mglPackageInit("main.js"); function connects the necessary files and the main.js game module.
You can specify null if you are writing a module in the current html file. Then use the code for the module. <script type='module'> Your code here </script>

gamer.js

I'll take a little more of your time before writing the game. Let's create a file gamer.js.
Its code will be: // Set your name gamer.projectName = "myGame"; // And versions gamer.projectVers = [ // New to up ["0.1", "25.05.2025 19:04", "Make project and main code"], ]; // Base gamer.base = { lang: "en", // Initial new player data level: 1, coins: 0, title: 'Noob!' update(){} }; // Translations gamer.langs = { LOADING_MESSAGE: { en: "Loading:", ru: "Загрузка:" } }; Gamer stores and operates the player's game data. Such as health, coins, current level and everything else that defines the player's unique features.
This class is responsible for loading and saving user data.

Here in projectName we specify the name of the project.
projectVers contains the version number, date and comment.

In gamer.base the initial values ​​for the player are stored. They can change during the game.
gamer.langs stores data for translating text into other languages.

main.js

It's time for adventure! Create a main.js

Its code will be: import * as THREE from 'three'; import {scene, camera, renderer, mglInitSections, mglApp} from 'mglcore/mgl.sections.js'; // App let myApp = new mglApp(); // Start myApp.onStartApp = function(){ // Make first cube const geometry = new THREE.BoxGeometry(); const material = new THREE.MeshBasicMaterial({ color: 0xfff000 }); this.mesh = new THREE.Mesh(geometry, material); scene.add(this.mesh); // Don't forget your camera! camera.position.set(0, 5, 5); camera.lookAt(0, 0, 0); } myApp.onAnimateApp = function(deltaTime){ this.mesh.rotation.x += 0.005; this.mesh.rotation.y += 0.005; } // Run myApp.runApp(); This is the simplest code imaginable.

1. Create a new mglApp instance.
2. Specify the game ready function onStartApp() with the cube code.
3. Rotate the cube in the animation function onAnimateApp().

And of course, don't forget to call runApp() to launch our application.

Your first game is ready! A rotating cube!
